# Imágenes para la Landing Page de Kelly

Por favor, coloca las siguientes imágenes en esta carpeta:

## Imágenes requeridas:

1. **pie.png** - Imagen flotante para la Sección 1 (Hero/Encabezado)
   - Tamaño recomendado: 400x600px o proporcional
   - Formato: PNG con fondo transparente

2. **sentada.png** - Imagen flotante para la Sección 2 (About/Información)
   - Tamaño recomendado: 350x500px o proporcional
   - Formato: PNG con fondo transparente

3. **fondo.jpg** - Imagen de fondo para la Sección 2
   - Tamaño recomendado: 1920x1080px o mayor
   - Formato: JPG/JPEG

4. **volando.png** - Imagen flotante para la Sección 3 (RSVP/Confirmación)
   - Tamaño recomendado: 350x500px o proporcional
   - Formato: PNG con fondo transparente

5. **mapa.png** - Imagen flotante para la Sección 4 (Ubicación)
   - Tamaño recomendado: 400x500px o proporcional
   - Formato: PNG con fondo transparente

## Características:

- Las imágenes flotantes tienen un efecto de parallax cuando haces scroll
- En versión móvil, las imágenes se ajustan automáticamente
- Los colores verdes de Tinkerbell se complementan con caracteres poéticos
- Las imágenes flotantes tienen sombreado suave para mejor visualización

¡Todas las imágenes deben estar optimizadas para web para mejor rendimiento!
